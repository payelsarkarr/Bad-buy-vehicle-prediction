# Bad Buy Vehicle Prediction Using Decision Trees

## Overview
This project builds a machine learning classification model to predict whether a vehicle should be considered a "kick" or bad buy. The analysis uses structured vehicle data and applies decision tree classification to identify patterns associated with poor purchase outcomes.

## Objective
The goal of this project is to develop a predictive model that helps identify high-risk vehicle purchases. By analyzing historical vehicle attributes, the model supports better decision-making and demonstrates how classification techniques can be applied to real-world business problems.

## Methodology
- Load and clean the vehicle dataset
- Perform exploratory analysis on relevant features
- Prepare the data for modeling
- Split the dataset into training and testing sets
- Train a Decision Tree Classifier
- Evaluate model performance using classification metrics

## Key Features
- Data cleaning and preprocessing
- Classification using decision trees
- Model evaluation with performance metrics
- Business-oriented interpretation of results
